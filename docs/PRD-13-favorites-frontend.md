# PRD-13: 收藏 & 互动前端

| 字段 | 内容 |
|------|------|
| 功能 | 详情页收藏按钮 + 首页展示互动数据 |
| 优先级 | P1 |
| 预估 | 0.5天 |

---

## 1. 产品目标

收藏 API 后端已实现，前端缺少交互入口。补齐收藏按钮和互动数据展示，激活用户参与感。

## 2. 详情页——收藏按钮

```
❤️ 收藏 (12)          ← 用 bp.favorite_count
   ↓ 点击
💖 已收藏 (13)         ← 用 bp.is_favorited
```

交互规则：
- 未登录点击 → 弹登录引导
- 已登录未收藏 → 发送 POST /api/blueprints/{id}/favorite
- 已收藏 → 发送 DELETE /api/blueprints/{id}/favorite
- 实时更新计数，乐观更新（先改UI再等API确认）

## 3. 首页/发现页——互动数据展示

每个图纸卡片上加：
```
┌────────────────────┐
│ [封面图]           │
│ 中世纪城堡         │
│ 👁 1.2k  💬 45  ❤️ 89 │  ← view_count / 评论数 / 收藏数
└────────────────────┘
```

### 数据来源

图纸列表 API 需返回：
- `view_count` → 已有字段
- `comment_count` → 需后端加
- `favorite_count` → 已有

## 4. 后端补充

**修改蓝图列表 API**：返回 `comment_count`

**浏览计数**：GET 详情时自动 +1（防刷：同用户同图纸 24h 内不重复计）

## 5. 验收标准

- [ ] 详情页收藏按钮可点击、状态切换正确
- [ ] 未登录点击收藏弹出登录引导
- [ ] 收藏数实时更新
- [ ] 首页/发现页卡片展示互动数据
- [ ] 详情页浏览自动+1（24h去重）
