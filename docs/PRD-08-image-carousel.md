# PRD-08: 图片轮播

| 字段 | 内容 |
|------|------|
| 功能 | 详情页多图轮播查看 |
| 优先级 | P1 |
| 预估 | 0.5天 |

---

## 1. 产品目标

图片上传已支持多张，但详情页目前只展示第一张。需要支持左右切换查看所有图片。

## 2. 交互设计

```
┌──────────────────────────────────┐
│         ← [1/4] →                │
│  ┌────────────────────────────┐  │
│  │                            │  │
│  │       主图 600x450         │  │
│  │                            │  │
│  └────────────────────────────┘  │
│  ┌───┐ ┌───┐ ┌───┐ ┌───┐      │
│  │ 1 │ │ 2 │ │ 3 │ │ 4 │  缩略图│
│  └───┘ └───┘ └───┘ └───┘      │
└──────────────────────────────────┘
```

**详情**：
- 左/右箭头或缩略图点击切换
- 左上角显示 N/M 计数
- 移动端支持滑动
- 无图或1张时不显示轮播控件
- 支持键盘 ← → 切换

## 3. 实现要点

### 前端 (`main.js`)

在 `showDetail()` 函数中修改图片展示区域：
- 从 `bp.images` 数组读取所有图片
- 维护 `currentImageIndex` 状态
- 渲染轮播控件

### CSS (`main.css`)

- 轮播容器 `.carousel`
- 缩略图 `.carousel__thumbnails`
- 导航按钮 `.carousel__nav`
- 响应式（移动端缩略图变小或隐藏）

**纯 JS，零依赖。**

## 4. 验收标准

- [ ] 多图显示轮播控件（箭头 + 缩略图）
- [ ] 左右切换正常
- [ ] 计数显示正确（如 1/4）
- [ ] 键盘 ← → 可切换
- [ ] 单图或无图不显示轮播控件
- [ ] 移动端触摸滑动支持
