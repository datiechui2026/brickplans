# PRD-11: 首页动态化

| 字段 | 内容 |
|------|------|
| 功能 | 首页从静态"最新图纸"升级为动态内容聚合 |
| 优先级 | P1（第二阶段） |
| 预估 | 1天 |

---

## 1. 产品目标

当前首页只展示最新图纸列表，信息单一。升级为内容聚合流，提高首页吸引力，引导用户探索。

## 3. 后端扩展

### 首页数据 API `GET /api/home`

返回聚合数据：
```json
{
  "featured": [...],       // 精选（按点赞数前3）
  "latest": [...],         // 最新上传（已有）
  "popular_tags": [...],   // 热门标签（使用次数前8）
  "active_users": [...],   // 近期活跃用户（最近上传）
  "stats": {
    "total_blueprints": N,
    "total_users": N
  }
}
```

## 4. 前端改造 (`main.js`)

- `renderHome()` → 从 `/api/home` 拉数据渲染
- 精选区：3卡片横滚（CSS scroll-snap）
- 最新区：保持现有卡片网格
- 热门标签：点击跳转发现页筛选
- 活跃用户：头像+用户名横滚

## 5. 验收标准

- [ ] `/api/home` 端点返回完整聚合数据
- [ ] 首页展示精选区块
- [ ] 热门标签可点击跳转
- [ ] 活跃设计师展示
- [ ] 移动端横滚流畅
- [ ] 搜索栏在首页可用
