# 积木品牌图纸下载服务调研报告

> 调研日期：2026-06-07
> 目的：了解国内积木品牌图纸下载服务现状，为 BrickPlan 产品定位提供参考

---

## 一、核心结论

**国内积木品牌基本不提供数字化图纸下载服务，市场存在明显空白。**

- 乐高是唯一拥有完善数字图纸体系的品牌（按套装编号免费下载 PDF + 3D 互动 App）
- 国产头部品牌（布鲁可、启蒙等）官网无任何图纸下载入口
- 少数品牌（双鹰/咔搭、沃马）官网有"下载"字样但功能简陋
- 中文互联网缺乏一个专门的积木图纸/MOC 分享社区

## 三、图纸获取方法汇总

### 方法 1：乐高官方（最标准）
```
步骤：
1. 找到套装编号（盒子正面/说明书封面 4-6 位数字）
2. 访问 lego.com/zh-cn/service/buildinginstructions
3. 输入编号 → 搜索 → 选择语言 → 下载 PDF
4. 或下载 LEGO Builder App 查看 3D 互动版
```

### 方法 2：BrickLink Studio
```
步骤：
1. 下载 BrickLink Studio 软件（免费）
2. 用户可自行设计 MOC 并导出图纸/零件清单
3. 内置 Gallery 浏览他人作品
```

### 方法 3：Rebrickable
```
步骤：
1. 注册 rebrickable.com
2. 浏览 MOC 作品
3. 免费图纸直接下载，付费图纸购买后下载
4. 可按自有零件筛选可搭建的作品
```

### 方法 4：国产品牌（有限）
- **沃马**：产品详情页有 "Download Free" 按钮 → 输入邮箱获取
- **双鹰/咔搭**：官网 → 下载服务 → 选择产品
- 绝大多数国产品牌：**只有包装盒内纸质说明书，无数字版**

### 方法 5：非官方渠道（不稳定）
- 贴吧/小红书/闲鱼：用户自发分享扫描版
- B站：搭建视频教程
- 微信群/QQ群：文件分享

## 四、市场机会分析

### 竞争格局

```
高质量
  ↑
  │  ★ BrickPlan (目标定位)
  │     ┌─ 中文优先
  │     └─ 好用的图纸分享体验
  │
  │  ★ Rebrickable (全球)
  │     ┌─ 最全的MOC库
  │     └─ 英文为主，国内访问慢
  │
  │  ★ 高砖 Gobricks
  │     ┌─ 有上传功能
  │     └─ 侧重零件销售
  │
  │  ★ Brick4 积木圈子
  │     ┌─ 数据库索引
  │     └─ 无上传功能
  │
  │  国产品牌官网 ××
  │  ┌─ 无图纸下载
  └──┴──────────────────→ 图纸下载能力
```

### BrickPlan 的差异化机会

1. **填补中文市场空白**：国内没有专门的积木图纸分享社区
2. **比 Rebrickable 更本地化**：中文界面 + 国内服务器 + 社交功能
3. **比高砖更专注图纸**：不做零件销售，专注 UGC 图纸分享和社区
4. **比 Brick4 更有交互**：不只是数据库，是社区 + 上传 + 互动
5. **采集策略**：
   - 种子内容：从 Rebrickable 精选免费 MOC 翻译引入（需注意版权）
   - 激励创作者：邀请国内 MOC 达人来上传作品
   - 品牌合作：未来可与拼奇、咔搭等品牌合作嵌入官方图纸
