## Variant: Clean Gallery

### Design stance
图片优先的编辑风格——像 Dribbble / Unsplash，让图纸本身成为视觉焦点。克制、优雅、呼吸感强。

### Key choices
- **Layout:** 非对称网格 + 大图 Hero（2/3 宽），强调视觉层次
- **Typography:** DM Serif Display（标题，有质感）+ Inter（正文，清晰现代）
- **Color:** 白色底 + 靛蓝点缀 (#6366F1)，极简调色盘
- **Interaction:** 细腻阴影 + 悬停上浮，按钮有微光晕

### Trade-offs
- **Strong at:** 高端感、图片展示效果极佳、浏览体验沉浸
- **Weak at:** 功能性信息（难度/零件数）不够显眼；LEGO 品牌感弱

### Best for
- 以浏览和发现为主的用户、对设计品质有要求的社区
