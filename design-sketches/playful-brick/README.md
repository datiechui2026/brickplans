## Variant: Playful Brick

### Design stance
LEGO 品牌风格——高饱和色彩、粗边框、俏皮圆角。让用户感觉"这是一个属于积木爱好者的游乐场"。

### Key choices
- **Layout:** 单栏内容流，Hero → 数据 → 热门 → 分类
- **Typography:** Fredoka One（标题）+ Nunito（正文），圆润友好
- **Color:** LEGO 经典色系（黄 #FFD500、红 #E3000B、蓝 #0051BA）
- **Interaction:** 粗边框卡片 + hover 弹跳效果，按钮有"按下"动效

### Trade-offs
- **Strong at:** 品牌辨识度极高，情感共鸣强，积木爱好者一眼爱上
- **Weak at:** 信息密度偏低，专业用户可能觉得不够高效；深色内容区略少

### Best for
- 面向大众积木爱好者、有社区氛围需求的场景
