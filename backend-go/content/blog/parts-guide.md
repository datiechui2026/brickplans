# 积木零件分类大全：从基础砖到特殊件，一篇看懂所有积木零件

> 刚入坑 MOC 的时候，看着图纸上的零件清单一脸懵："1x2 砖"和"1x2 板"有什么区别？"Tile"和"Plate"又是什么？

别担心，这篇指南将带你系统了解积木零件的分类体系。**掌握了这些，你看图纸的能力会提升 80%。**

---

## 一、为什么要了解零件分类？

了解零件分类不只是为了"认识名字"，更重要的是：

1. **快速选零件**：搭建时知道去哪里找需要的零件
2. **看懂图纸**：零件清单和步骤图不再像天书
3. **优化设计**：选择合适的零件类型，让作品更稳固、更美观
4. **购买零件**：在二手平台或零件店准确表达需求

## 二、零件分类体系（6 大类）

### 1️⃣ 基础砖（Brick）

**特征**：高度 = 3 个板的厚度，是积木搭建的"主力军"

| 常见类型 | 样子 | 用途 |
|---------|------|------|
| 1x1 砖 | 一颗小方块 | 填充、装饰、纹理细节 |
| 1x2 砖 | 两颗宽 | 最常用的基础砖 |
| 1x4 砖 | 四颗宽 | 墙体、梁柱结构 |
| 1x6 / 1x8 砖 | 更长 | 大跨度结构 |
| 2x2 砖 | 方形 | 墙角、承重 |
| 2x4 砖 | 标准砖 | 乐高最经典的零件 |

**MOC 技巧**：长砖（1x6/1x8）比多个短砖拼接更稳固，但短砖可以做出凹凸纹理。

---

### 2️⃣ 板（Plate）

**特征**：高度 = 1/3 个砖的厚度，是细节调整的"魔法师"

| 常见类型 | 用途 |
|---------|------|
| 1x1 板 | 微调高度、填充缝隙 |
| 1x2 板 | 最常用的板 |
| 1x4 / 1x6 板 | 墙面装饰、屋顶 |
| 2x2 板 | 方形填充 |
| 2x4 板 | 地板、底座 |
| 2x6 / 2x8 板 | 大面积底板 |

**MOC 技巧**：3 块板叠起来 = 1 块砖的高度。这个"3:1"比例是 MOC 中最常用的调整技巧。

---

### 3️⃣ 光面板（Tile）

**特征**：顶部没有凸点（Stud），表面光滑

**用途**：
- 地板、桌面、路面
- 装饰墙面（光滑表面更真实）
- 文字/图案印刷的载体
- 人仔脚下的"站立面"

**常见规格**：1x1 Tile、1x2 Tile、1x4 Tile、2x2 Tile、2x4 Tile

**MOC 技巧**：建筑类 MOC 中，用光面板做地板比用普通砖更真实。赛博朋克场景中，光面板 + 透明零件 = 霓虹灯效果。

---

### 4️⃣ 斜坡砖（Slope）

**特征**：带有倾斜角度的砖，用于屋顶、车头、翅膀等

| 常见类型 | 角度 | 用途 |
|---------|------|------|
| 斜坡砖 30° | 30度 | 屋顶、坡道 |
| 斜坡砖 45° | 45度 | 车头、城堡屋顶 |
| 反向斜坡砖 | 倾斜方向相反 | 建筑屋檐底部 |
| 弧形斜坡砖 | 曲面 | 圆润造型 |

**MOC 技巧**：城堡的屋顶用 45° 斜坡砖，现代建筑用 30° 斜坡砖，更符合真实比例。

---

### 5️⃣ 科技件（Technic）

**特征**：带有孔洞和销钉的零件，用于机械结构和可动设计

**核心零件**：
- **科技砖/梁**：带孔的梁状零件，骨架结构
- **销钉**：连接科技梁的连接件
- **齿轮**：传动和转向
- **轴**：旋转运动传递
- **连杆**：平行四边形机构
- **履带**：坦克/挖掘机等

**MOC 技巧**：机甲类 MOC 必须用科技件做关节，纯基础砖搭建的关节支撑力不足。

---

### 6️⃣ 特殊件（Special Element）

**特征**：无法归入以上类别的"异形件"，用途专一

**常见类型**：

| 类型 | 例子 | 用途 |
|------|------|------|
| 门/窗框 | 各种尺寸的门窗 | 建筑门窗 |
| 栅栏/栏杆 | 1x2x2 栅栏 | 阳台、围栏 |
| 轮子/轮胎 | 各种尺寸 | 车辆 |
| 植物件 | 树叶、花朵、树木 | 自然场景 |
| 人仔配件 | 武器、工具、帽子 | 人仔搭配 |
| 印刷件 | 带图案的砖/板 | 装饰、标识 |
| 透明件 | 各种透明颜色的砖 | 窗户、灯效、魔法 |

## 三、零件颜色指南

积木零件的颜色体系非常丰富，以下是常用颜色分类：

| 色系 | 常见颜色 | 用途场景 |
|------|---------|---------|
| 中性色 | 白色、黑色、灰色、深灰 | 建筑主体、机甲、车辆 |
| 暖色 | 红色、橙色、黄色、棕色 | 城堡、车辆、火焰 |
| 冷色 | 蓝色、浅蓝、绿色、紫色 | 水景、科幻、赛博朋克 |
| 特殊色 | 透明色、珍珠色、金属色 | 窗户、特效、装饰 |

**MOC 技巧**：新手建议先用 3-4 种颜色搭配，避免配色混乱。经典配色方案：
- 建筑：白+灰+棕（自然系）
- 机甲：灰+黑+红（战斗系）
- 科幻：银灰+蓝+透明蓝（科技系）

## 六、常见问题

**Q：怎么区分"砖"和"板"？**
A：看厚度。砖 = 3 个板的高度。最简单的判断方法：砖侧面有 3 条凸起纹路，板只有 1 条。

**Q：为什么我的零件拼不紧？**
A：可能是零件批次问题（不同品牌模具精度不同），也可能是零件老化。建议同一品牌混用。

**Q：什么是 SNOT？**
A：SNOT = Studs Not On Top（凸点不在上面），是一种让零件朝侧面搭建的技巧，做建筑外墙和机甲细节时非常常用。
