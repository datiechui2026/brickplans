# BrickPlan 新手教程：注册、找图纸、下载、搭建全流程指南

欢迎来到 BrickPlan——中文积木 MOC 图纸分享社区！

无论你是刚接触积木的新手，还是经验丰富的 MOC 玩家，这篇教程都会带你快速上手 BrickPlan 的所有功能。

---

## 一、什么是 BrickPlan？

**BrickPlan**（积木图纸分享社区）是一个专注积木 MOC 图纸的中文平台。你可以在这里：

- **浏览**海量 MOC 图纸（建筑、车辆、机甲、奇幻、科幻、场景）
- **下载**图纸 PDF 和零件清单
- **上传**自己的 MOC 作品，分享给其他玩家
- **互动**——点赞、收藏、评论、关注你喜欢的创作者

---

## 二、注册账号

### 步骤 1：进入网站

打开浏览器，访问 **https://brickplan.cn**（或服务器 IP + 端口）。

### 步骤 2：注册

1. 点击右上角「注册」按钮
2. 输入用户名（2-20 个字符，中英文均可）
3. 输入邮箱地址
4. 设置密码（至少 8 位）
5. 点击「注册」完成

### 步骤 3：完善个人资料

注册后，建议补充：
- **头像**：让你的账号更有个性
- **个人简介**：介绍一下你的搭建偏好（建筑？机甲？科幻？）
- **个人主页**：其他玩家点击你的名字可以看到你的作品合集

---

## 三、寻找图纸

### 方法 1：分类浏览

在首页或「探索」页面，图纸按 6 个分类展示：

| 分类 | 典型作品 | 适合人群 |
|------|---------|---------|
| 🏛️ 建筑 | 城堡、别墅、神社 | 建筑爱好者 |
| 🚗 车辆 | 赛车、卡车、机车 | 车模爱好者 |
| 🤖 机甲 | 突击机甲、忍者机甲 | 机甲粉丝 |
| 🐉 奇幻 | 巨龙、独角兽、矮人铁匠铺 | 奇幻题材 |
| 🚀 科幻 | 星际战舰、太空基地 | 科幻迷 |
| 🌴 场景 | 海盗湾、便利店、游乐场 | 场景控 |

### 方法 2：搜索

在搜索框输入关键词，如 "城堡"、"机甲"、"赛博朋克"，系统会匹配标题和描述。

### 方法 3：按难度筛选

每份图纸都标注了 **难度星级**（1-5 星）：

| 难度 | 含义 | 推荐人群 |
|------|------|---------|
| ⭐ | 新手入门（300-500 零件） | 零基础 |
| ⭐⭐ | 简单（500-1000 零件） | 有基础 |
| ⭐⭐⭐ | 中级（1000-2500 零件） | 中级玩家 |
| ⭐⭐⭐⭐ | 中高级（2000-4000 零件） | 经验丰富 |
| ⭐⭐⭐⭐⭐ | 专家级（4000+ 零件） | 硬核玩家 |

> **新手建议**：从 ⭐ 或 ⭐⭐ 的图纸开始，先体验完整的搭建流程。

---

## 四、查看图纸详情

点击图纸卡片进入详情页，你会看到：

### 基本信息
- **标题**：图纸名称
- **作者**：创作者的名字
- **分类**：所属类别
- **难度**：1-5 星
- **零件数**：精确的零件数量
- **发布时间**：图纸上传日期

### 内容区域
- **图片展示**：多张成品图，展示各个角度
- **描述**：详细介绍作品的搭建难点、零件特色和适合人群
- **零件清单**：按类别列出所有需要的零件

### 互动区域
- **点赞**：表达你的喜欢
- **收藏**：保存到你的收藏夹
- **评论**：向作者提问或分享你的感受
- **分享**：复制链接分享给朋友

---

## 五、下载图纸

### 下载方式

1. 在图纸详情页找到「下载图纸」按钮
2. 点击后会自动下载 PDF 格式的搭建说明书
3. 同时提供零件清单（BrickLink 格式，方便购买零件）

### 图纸格式说明

| 格式 | 内容 | 使用方式 |
|------|------|---------|
| PDF | 步骤图 + 零件清单 | 打印或在电脑/平板上查看 |
| 零件清单 | 零件编号 + 数量 | 用于购买零件 |
| Studio 文件（可选） | 3D 模型文件 | 在 BrickLink Studio 中打开 |

> **温馨提示**：图纸版权归原作者所有，请勿未经许可二次上传或商用。

## 六、开始搭建

### 准备工作

1. **清点零件**：根据零件清单核对是否有缺少
2. **准备工具**：零件分拣盒（按颜色或类型分类）
3. **选择场地**：平整的桌面，充足的光线
4. **打开图纸**：平板或电脑上看图纸最方便

### 搭建技巧

- **按步骤来**：不要跳步，每步确认零件位置正确
- **分模块搭建**：大型作品建议分模块完成，最后组装
- **遇到问题**：先看步骤图下方的说明，多数问题图纸上都有标注
- **求助社区**：在评论区提问，作者或其他玩家会帮你解答

---

## 七、上传你的作品

搭建完成后，你也可以把自己的 MOC 作品分享给社区！

### 上传步骤

1. 注册并登录账号
2. 点击右上角「上传」按钮
3. 填写作品信息：
   - **标题**：给你的作品起个名字
   - **描述**：介绍作品的设计思路、搭建难点
   - **分类**：选择合适的分类
   - **难度**：评估 1-5 星
   - **零件数**：填写精确数量
   - **标签**：添加关键词标签，方便搜索
4. 上传图片（至少 1 张，建议 3-5 张不同角度）
5. 上传图纸 PDF 或 Studio 文件
6. 点击「发布」

### 上传技巧

- **封面图**至关重要：选择最漂亮的角度作为封面，点击率提升 50%
- **描述要详细**：搭建难度、零件特色、成品展示、适合人群——写清楚其他玩家才敢下载
- **标签要精准**：如"赛博朋克""街景""新手友好"，方便搜索

## 九、常见问题

**Q：下载的图纸打不开怎么办？**
A：PDF 文件请使用浏览器或 PDF 阅读器打开。Studio 文件需要安装 BrickLink Studio 软件。

**Q：零件不够怎么办？**
A：BrickPlan 的每份图纸都附带零件清单，你可以按清单在淘宝、闲鱼或高砖购买所需零件。

**Q：我能上传别人的作品吗？**
A：请只上传你自己的原创作品。转载他人作品需获得原作者授权并注明出处。

**Q：图纸有错误怎么办？**
A：在评论区留言告知作者，作者会尽快修正并重新上传。
